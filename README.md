# FX Clearing Hub Website

A Next.js website for FX Clearing Hub, providing payment solutions for high-risk merchants.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Pages

- Home: Overview with hero, industries, solutions, etc.
- Solutions: Links to specific solutions
- Industries: List of industries served
- High-Risk Merchant Account: Details on merchant accounts
- Forex Payment Processing: Forex-specific solutions
- Gaming Payment Solutions: Gaming payment details
- About: Company information
- Contact: Contact form
- Blog: Blog posts (starter topics)

## Technologies

- Next.js 14
- TypeScript
- Tailwind CSS
